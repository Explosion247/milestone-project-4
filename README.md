# Milestone project 4

## Deployment

create a readme page

Create a virtual environment

pip3 install django

django-admin startproject milestone-project-4

create gitIgnore

python3 manage.py migrate

python3 manage.py createsuperuser

Create a gitHub repository using VS code (Link your github acount with VS studio code, then open the source control pannel. Using the Initialise Repository, choose whether you want your repository to be public or private, and  this will creat your repository)

create heroku (after logging into heroku, open the projects page and click new at the top right of the page, and create a new app. Pick a name that is suitable and available, then choose your region, either the US or Europe (i choose europe) and then create at the bottom.)

pip3 install django-allauth==0.50.0
